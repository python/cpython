######################
  Python 教程
######################

Python 是一种易于学习且功能强大的编程语言。 它提供了高效的高级数据结构和简单有效地面向对象编程的方法。
Python 的优雅语法和动态类型，以及解释型语言的本质，使它成为多数平台上写脚本和快速开发应用的理想语言。

到 Python 官网 https://www.python.org/ 可以免费获取Python 用于各个主要系统平台的Python解释器及其丰富的标准库的源码或机器码，并可自由地分发。而且该网站提供了许多免费的第三方 Python 模块、程序、工具和它们的文档。

Python 解释器可以轻易地使用 C 或 C++（或者其他可以通过 C 调用的语言）扩展新的功能和数据类型。
Python 同样适用于定制软件中的扩展程序语言。

本教程非正式地向读者介绍了Python语言和系统的基本概念和特性。最好在阅读的时候准备一个 Python 解释器进行练习。所有的例子都是相互独立的，因此该教程也可离线阅读。

有关标准的对象和模块，请参阅 :ref:`library-index`。
:ref:`reference-index` 提供了更为正式的语言定义。  想要编写 C 或者 C++ 扩展可以参考 :ref:`extending-index` 和
:ref:`c-api-index`。 此外还有不少书籍深入讲解 Python。

该教程不试图完整地介绍每一个功能，甚至可能没有完全涉及常用功能。相反它只介绍 Python 中最值得注意的功能，以及让你体会到这个语言的风格特色。学习完这个教程，你可以阅读和编写 Python 模块和程序，也可以继续学习 :ref:`library-index`。

The :ref:`glossary` 也很值得一读。
